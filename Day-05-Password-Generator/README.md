# Password Generator (Day 5)

A Python CLI application that generates a secure random password based on user preferences.

## Objective
To practice Python concepts like loops, lists, and randomization while building a useful utility.

## How It Works
- User specifies:
  - Number of letters
  - Number of numbers
  - Number of symbols
- Program randomly selects characters
- Shuffles them to create a secure password

## How to Run
```bash
python main.py
