# Band Name Generator (Day 1)

A simple Python program that generates a fun band name based on user input.

## Objective
Combine user inputs (city name and pet name) to create a unique band name.

## How It Works
- User enters the name of the city they grew up in
- User enters the name of their pet
- Program combines both to generate a band name

## How to Run
```bash
python main.py
