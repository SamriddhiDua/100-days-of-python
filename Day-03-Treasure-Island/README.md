# Treasure Island Game (Day 3)

A text-based adventure game built using Python where the player makes choices to find the treasure.

## Objective
To practice conditional statements and decision-making logic in Python.

## How It Works
- Player is presented with multiple choices
- Each decision affects the outcome
- Some paths lead to "Game Over"
- One correct path leads to winning the treasure

## How to Run
```bash
python main.py
