# Tip Calculator (Day 2)

A simple Python CLI application to calculate the total bill including tip and split it among multiple people.

## Objective
To practice basic Python concepts like user input, arithmetic operations, and type casting.

## How It Works
- User enters total bill amount
- User selects tip percentage (e.g., 10%, 12%, 15%)
- User enters number of people
- Program calculates and displays amount per person

## How to Run
```bash
python main.py
